Cloud Infrastructure Engineer. Join Amazon Web Services and help build the cloud infrastructure that powers the internet! We're seeking an experienced Cloud Infrastructure Engineer to design and maintain scalable AWS systems.

What You'll Do: Design cloud architectures (servers, databases, networking, security). Write infrastructure-as-code using Terraform and CloudFormation. Monitor systems 24/7 and respond to incidents. Deploy updates, manage scaling, optimize costs. Work with DevOps tools (Docker, Kubernetes, CI/CD pipelines). Stay current with new AWS services and features.

Requirements: Bachelor's degree in Computer Science or related field. 3+ years cloud infrastructure experience. AWS certifications required (Solutions Architect, DevOps Engineer). Experience with Terraform, Kubernetes, Docker. Strong Linux and networking knowledge. Must pass rigorous technical interviews. Willingness to work on-call rotations and 50+ hour weeks. Must relocate to Seattle (no remote option).

Compensation: $130,000 - $180,000 per year. Benefits: Competitive salary and equity. Comprehensive health benefits. 401(k) matching. Relocation assistance.

Constant learning required - AWS releases new services weekly. High-stakes environment where production outages impact millions of users.
